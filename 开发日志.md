### 阶段二：基础工具集
**目标**：实现核心工具（Read、Bash、Write、Edit、Glob、Grep）
**状态**：✅ 已完成 (2026-04-03)

- [x] Read 工具 - 支持 UTF-8 编码、offset/limit 行读取
- [x] Write 工具 - 支持创建、覆盖，自动创建父目录
- [x] Edit 工具 - 精确字符串替换，自动处理换行符差异
- [x] Glob 工具 - 文件模式匹配，支持递归搜索
- [x] Grep 工具 - 正则表达式搜索，支持上下文行
- [x] Bash 工具 - 跨平台 Shell 命令执行，支持超时

**工具 API 示例**：
```python
from mini_claude.tools import ReadTool, WriteTool, BashTool

# Read 工具
read_tool = ReadTool()
result = await read_tool.execute(file_path="README.md", offset=1, limit=10)

# Write 工具
write_tool = WriteTool()
result = await write_tool.execute(file_path="test.txt", content="Hello!")

# Bash 工具
bash_tool = BashTool()
result = await bash_tool.execute(command="echo hello", description="Greet")
```
