# MiniClaude 开发计划

## 整体框架

```
mini_claude/
├── pyproject.toml              # 项目配置
├── src/
│   └── mini_claude/
│       ├── __init__.py
│       ├── __main__.py         # CLI 入口
│       ├── core/               # 核心抽象
│       │   ├── __init__.py
│       │   ├── tool.py         # Tool 基类
│       │   └── command.py      # Command 基类
│       ├── config/             # 配置管理
│       │   ├── __init__.py
│       │   └── settings.py
│       ├── models/             # 数据模型
│       │   └── __init__.py
│       ├── utils/              # 工具函数
│       │   ├── __init__.py
│       │   ├── git.py
│       │   ├── env.py
│       │   └── format.py
│       ├── tools/              # 工具实现
│       │   └── __init__.py
│       ├── commands/           # 命令实现
│       │   └── __init__.py
│       └── services/           # 服务层
│           └── __init__.py
```
